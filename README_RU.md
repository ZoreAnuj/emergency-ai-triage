## Миссия


Перевод клинических алгоритмов и протоколов в надёжные, проверяемые цифровые инструменты, которые помогают врачам и пациентам в критических ситуациях.


---


## Digital WM Core — обзор


**FDT (инструменты фотодинамической терапии)** — клинические калькуляторы, планирование сессий, модули моделирования.


**EMS (скорая и экстренная медицина)** — быстрый доступ к протоколам, справочникам, инструментам триажа.


**PANIC (помощь при панических атаках)** — офлайн-приложения и ассистенты для стабилизации, дыхательных техник и само-помощи.


---


## Основные проекты (выборочно)


- **Panic Helper (iOS, SwiftUI)** — клиническое пособие для стабилизации при панике.
- **EMS Stepspeed** — потоки принятия решений для реанимационно-экстренных сценариев.
- **FDT Clinical Assistant** — расчёты доз и планирование сессий ФДТ.
- **Universal REST Template** — шаблоны микросервисов (Rust / Kotlin / Node).
- **MedKey Auth Service** — сервис авторизации и управления API ключами.


---


## Архитектура (высокоуровнево)


- API Gateway → Auth Service → Микросервисы (FDT, EMS, PANIC) → Worker / Task Queue → Storage / Metrics
- AI/ML модули запускаются как изолированные сервисы и предоставляют предсказания через защищённые API.
